# 正态分布是什么

  • 用自己的话重新描述，而不是复制
  • 尝试思考与它相似的内容有哪些

正态分布又叫作高斯分布，是指一个事物中间聚集，两端稀少的经典分布。大量的实践和理论分析表明，绝大多数随机变量均服从这个分布。

比如人的身高、成绩、误差率、投资收益率。

这种数学分布揭露了一个现实：你不会特别差，也不会特别好。这世界上只有 1%的两极分化，大多数都是呈现中等水平。

道理上很容易理解，但很多人拒绝接受现实。认为不均匀就是不公平，甚至举着“正义”的大旗为所欲为。

# 为什么正态分布能起作用

  • 原理是什么
  • 前提是什么

# 如何用正态分布解决问题

  • 知识的可操作性（如何用这个知识）
    • 结构化（Structured）
    • 流程化（SOP）
    • 可视化（Viusal）
    • 工具化（Tools）
  • 需要遵循的原则（Principle）
  • 后续行动计划（Plan）

# 可以解决哪些具体问题

  • 适用哪些场景
  • 适用哪些现象
  • 对他人有什么价值

# 经验实践

  • 过程中的感受是怎样的（Feel）
  • 按这个知识去做了，结果怎么样（Result）
  • 有哪些重要的定量知识（How many）
